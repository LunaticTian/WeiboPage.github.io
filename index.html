微博参数说明

UserInfo

记录微博用户的信息表

  • screen_name 用户名称
  • statuses_count 发送微博总量
  • verified_reason 个人说明
  • description 个人介绍
  • followers_count 粉丝数量
  • follow_count 关注数量
  • count 在2018年度发送的微博总数
  • profile_url 主页地址

ContentInfo

记录微博用户所发的微博

  • screen_name 用户名称
  • subject_url 微博地址
  • Time 微博发送时间
  • reposts_count 转发数
  • comments_count 评论数
  • attitudes_count 点赞数
  • text 微博内容(包含转发的内容)
  • status 微博类型(原创/转发/视频)

CommentInfo

记录微博内容评论的内容

  • head_User 被评论的用户
  • User评论用户
  • url 微博评论地址
  • text 评论内容
  • created_at 评论时间
